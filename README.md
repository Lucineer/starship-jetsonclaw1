# starship-jetsonclaw1

> Jack into the bridge. Every system is real telemetry.

A MUD-style TUI where you walk the starship and see actual Jetson hardware.

## Rooms

| Room | Real System |
|---|---|
| Bridge | CPU load, uptime, fleet comms |
| Engine Room | GPU cores, CUDA kernels |
| Life Support | 9 thermal zones |
| Cargo Bay | Unified memory, NVME |
| Sickbay | Process health |
| Holodeck | Seed-2.0-Mini creative engine |
| Science Lab | GPU perception kernel |
| Airlock | Network interfaces |
| Quarterdeck | Captain log |

## Usage

```
python3 starship-jetsonclaw1.py
```

Every number is real. Not a simulation.
